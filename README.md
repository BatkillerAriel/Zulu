# Zulu app



## Running the backend
1. Make sure you have mkvirtualenv installed
2. Run
```bash
mkvirtualenv -p python3.8 zulu
cd backend
pip install -r requirements.txt
pip install -r test-requirements.txt
```

### Running backend tests
Run:
```bash
pytest -vvv -s tests/
```
